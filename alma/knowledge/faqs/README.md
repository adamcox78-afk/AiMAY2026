# Alma Knowledge Base

This is Alma's Obsidian-compatible knowledge vault.

## Structure
- `faqs/` - Frequently asked questions and answers
- `daily_briefs/` - Archived daily briefings
- `integrations/` - Notes on connected services
- `research/` - Research reports from the Research Agent

## How to Use
All files are standard Markdown compatible with Obsidian.
Alma automatically saves important information here.
Tags follow Obsidian format: #topic/subtopic
